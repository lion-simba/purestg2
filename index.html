purestg2

Stargazer authentication module for linux PPP daemon

Download .zip Download .tar.gz View on GitHub

Stargazer (http://stg.dp.ua) is a powerful billing system. PPPD is a famous PPP daemon for Linux systems.

Purestg2 is a bridge between Stargazer and pppd. Purestg2 allows you to authorize PPP connecting users with Stargazer.

More detailed information can be found in README.

Discussion in Russian and Ukrainian: http://local.com.ua/forum/topic/25760-purestg2/
